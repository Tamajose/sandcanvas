# SandCanvas
SandCanvas is a full-stack interactive web application where users create digital artwork by pouring simulated sand with different colors, textures, and tools. Inspired by modern generative art platforms, SandCanvas combines physics-based particle simulation with a clean, creative user experience.

# Overview
SandCanvas allows users to:
- Pour digitally simulated sand onto a canvas
- Control colors, speed, and grain size
- Use creative tools to shape artwork
- Save, share, and explore creations through user accounts

# Key Features
* Sand Simulation Canvas
  - Click or hold to pour sand
  - Real-time falling behavior using Canvas/WebGL
  - Smooth physics simulation for natural sand flow
* Color Palette & Tools
  - RGB color picker and custom gradients
  - Adjustable sand speed and grain size
  - Dual sand instances (cursor-based and fixed)
  - Eraser tool for removing sand
  - Shovel tool for scooping sand
  - Wind simulation affecting sand physics
  - Structures that influence sand movement
  - Dark mode with neon colors
* User Accounts & Authentication
  - Secure signup/login system
  - Password hashing
  - Personal gallery for each user
